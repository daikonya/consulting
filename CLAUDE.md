**1. 事業の背景・課題**
```
【現状】
熊本県阿蘇郡小国町に位置する御宿だいこんや（丸正旅館）において、
2025年12月〜2026年5月の予約データを分析した結果、以下の現状が確認された。

・全予約室数に占める直販（公式サイト経由）の割合は3〜11%に過ぎず、
　残り90%超をOTA（楽天トラベル・じゃらんnet・一休.com等）に依存している
・OTA手数料（15〜20%/件）の負担が収益を継続的に圧迫している
・OTA経由予約では顧客データが取得できないため、
　リピーター育成施策や顧客セグメント別のアプローチが実施できていない
・訪日外国人向けの集客体制が未整備で、インバウンド予約は月0〜2件にとどまる
・LINE・SNS等のデジタル接点が未整備で、潜在顧客への能動的なアプローチができていない
・Webサイトのアクセスデータ・予約データの統合分析が行えておらず、
　施策改善のPDCAサイクルが機能していない

【課題】
① 直販予約を獲得するためのデジタル基盤（公式LP・予約動線）が未整備
② 顧客データの取得・蓄積・活用ができていない
③ インバウンド向け多言語コンテンツおよび情報発信が不足
④ LINE・SNSによる集客・ファン化・リピーター育成が未着手
⑤ データ分析に基づく効果測定・改善サイクルが確立されていない
```

**2. 事業目的（目指す姿）**
```
3年後に、公式Webサイト・LINE・SNSを基盤とした
「データで意思決定できる宿泊施設運営」を実現する。

・直販率を現状の約7%から30%以上に引き上げ、OTA手数料コストを削減する
・LINE公式アカウントを通じた顧客接点を確立し、リピーター率・客単価を向上させる
・多言語コンテンツ・SNS発信を強化し、訪日外国人旅行者の受入体制を整備する
・GA4・BIツールによるデータ分析を月次で実施し、施策改善を継続的に行う
・本事業で構築したDXモデルをTOJISMのコンサルティングを通じて
　阿蘇エリアの他宿泊施設へ展開し、地域一体の観光DXを推進する
```
